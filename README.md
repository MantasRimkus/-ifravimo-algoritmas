# AES Šifravimo Sistema

## Projekto aprašymas
Šis projektas yra žiniatinklio sistema, skirta tekstui šifruoti ir dešifruoti naudojant AES (Advanced Encryption Standard) algoritmą. Sistema sukurta kaip universiteto užduotis.

## Funkcionalumas
- Teksto šifravimas ir dešifravimas naudojant AES algoritmą
- Šifravimo modų pasirinkimas:
  - ECB (Electronic Codebook)
  - CBC (Cipher Block Chaining)
  - CFB (Cipher Feedback)
- AES rakto ilgiai:
  - 128 bitų
  - 192 bitų
  - 256 bitų
- Slapto rakto įvedimas
- Šifruotų duomenų išsaugojimas į `.txt` failą
- Šifruotų duomenų nuskaitymas iš `.txt` failo
- Patogi grafinė vartotojo sąsaja

## Naudotos technologijos
- HTML5
- CSS3
- JavaScript
- CryptoJS biblioteka
